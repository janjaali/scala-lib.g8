# scala-lib.g8

[Giter8] skeleton for scala libraries.

## Usage

Giter the template

```shell
g8 janjaali/scala-lib.g8
```

## Test template locally

To test the template locally create a new directory and run:

```shell
mkdir testDir
cd testDir
g8 file://../scala-lib.g8/
```

[Giter8]: http://www.foundweekends.org/giter8/
